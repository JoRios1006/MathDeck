## Things I'd still improve

### 1. Sign charts are wrong for repeated roots

This is the biggest remaining mathematical issue.

Example:

```
x⁴ + 2x³ -39x² -40x +400
```

The roots listed are

```
-5
4
```

Yet it's a quartic, so unless there are complex roots, those roots must have even multiplicity. In fact,

[
x^4 + 2x^3 -39x^2 -40x +400 = (x+5)^2(x-4)^2.
]

A square is **never negative**.

Your card says:

```
Positive

(-∞,-5)
(-5,4)
(4,∞)

Negative

(none)
```

The "Negative: none" part is correct, but the positive intervals should really be:

```
(-∞,-5)
(-5,4)
(4,∞)

Zeros:
{-5,4}
```

or simply note that the function is positive everywhere except at the roots. Right now the interval listing is technically okay because the roots are excluded, but it's missing the important insight that the sign does **not** change across an even-multiplicity root.

I would eventually include multiplicities.

### 2. Root multiplicities

Instead of

```
Roots

-5
4
```

I'd print

```
Roots

-5 (×2)
4 (×2)
```

That immediately explains the sign chart.

## Minor cosmetic suggestions

Instead of

```
Positive

...
```

I'd write

```
f(x) > 0
```

Likewise

```
f(x) < 0
```

It feels more mathematical.

---

Instead of

```
Roots
```

I'd probably use

```
Zeros
```

Both are correct, but "zeros of the function" is common terminology once you start talking about arbitrary functions.

---

## One thing I absolutely love

You're generating functions **from their roots**.

That was the right design decision.

It guarantees

* nice coefficients
* nice factors
* meaningful derivatives
* meaningful sign charts

instead of random garbage.

That's exactly how I'd have built the engine.

---

## Final verdict

This is no longer "a Haskell script."

It's the beginning of a legitimate math practice generator.

I'd happily put this on GitHub as **MathDeck v0.2**.

The next big milestone I'd work toward isn't more algebra. It would be **card diversity**:

* "Find the derivative."
* "Find the roots."
* "Determine where (f(x) > 0)."
* "Compute the indefinite integral."
* "Evaluate (\lim_{x\to a} f(x))."
* "Complete function analysis."

Using the same generated function for multiple card types would dramatically increase the educational value without making the generator much more complicated. That's where I think MathDeck starts becoming a genuinely compelling study tool rather than just a CSV exporter.
